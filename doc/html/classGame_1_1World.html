bool Game::World::loadRoom ( const std::string &  _fileName)

World::loadRoom Loads the specified '*.room' into the World's current active Room and will attempt to handle common file handling exceptions if loading fails. World's currently active Room will not be modified if the room could not be loaded correctly.

Exceptions will be automatically echoed to std::cerr

Parameters
_fileNameThe file containing room data
Returns
True if the World's current active Room was updated, false if the file was not loaded for some reason.
